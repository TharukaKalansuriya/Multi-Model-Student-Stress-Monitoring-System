## 📞 **Troubleshooting**

### Theme doesn't switch
- Check SharedPreferences is initialized
- Restart the app
- See: UI_REDESIGN_GUIDE.md → Theme Provider section

### Circular bars don't animate
- Ensure SingleTickerProviderStateMixin is used
- Check animation duration (1.5s default)
- See: QUICK_REFERENCE.md → Animation timings

### Settings icon not visible
- Check AppBar configuration
- Verify icon is enabled
- See: VISUAL_GUIDE.md → Settings section

### Scores showing as 0
- Start collection first
- Wait for data to be collected
- Then check stress level
- See: COMPLETION_SUMMARY.md → Getting Started
